HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos
HTML podem ser interpretados por navegadores.

HTML é a abreviação para a expressão inglesa HyperText Markup Language, que significa
Linguagem de Marcação de Hipertexto.
